# Changelog

## Version 2.0.0

### New Features
- **Multi-language Support**: Added support for 15 languages (English, Spanish, French, Mandarin, Japanese, Finnish, Afrikaans, Swahili, German, Portuguese, Italian, Russian, Arabic, Hindi, Korean)
- **Glossy 3D Button Design**: Enhanced button styling with glossy 3D effect and press animations
- **Customizable Background Finishes**: Three background options (Basic, Matte, Plastic) for personalized appearance
- **Click Sound Effects**: Added audio feedback on button press for better user experience
- **Undo Button**: Added undo functionality for error correction
- **Improved Tip Calculator**: Enhanced tip calculator with breakdown display showing tip amount separately

### UI Improvements
- **Redesigned Button Layout**: Swapped positions of UNDO and ENTER buttons for better ergonomics
- **Enhanced Button Styling**: Improved button appearance with glossy 3D effect and top-left highlight
- **Improved Dark Mode**: Better display color contrast in dark mode for improved readability
- **Cleaner Interface**: Removed header for more screen space and cleaner look
- **Settings Button**: Moved settings button to top-right corner for easier access

### Technical Improvements
- **Significant Figures Limiting**: Limited display to 12 significant figures for better readability
- **Improved Number Formatting**: Enhanced number formatting with better decimal handling
- **Enhanced Accessibility**: Improved accessibility features and support
- **Better Error Handling**: More robust error handling throughout the app

### Bug Fixes
- Fixed decimal point button functionality
- Improved tip calculation rounding to 2 decimal places for currency accuracy
- Fixed display text color in dark mode
- Various UI refinements and optimizations

## Version 1.0.0

- Basic calculator operations (addition, subtraction, multiplication, division)
